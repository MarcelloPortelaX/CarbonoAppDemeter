# Referências visuais

## Arquivos integrais

- `demeter-carbono-light.png`: composição completa em modo claro.
- `demeter-carbono-dark.png`: composição completa em modo escuro.
- `demeter-brand-symbol.png`: símbolo principal fornecido para o aplicativo; usado sem redesenho no ícone, splash e interface.

## Recortes por tela

A pasta `screens/` contém recortes com moldura e recortes `*-screen.png` apenas da área útil. Os recortes de área útil são usados por `scripts/visual_diff.py`.

As imagens são especificações de aparência. Não use OCR para copiar texto deformado: use os textos definidos em `design/specs/COPY_GUIDE.md`.
