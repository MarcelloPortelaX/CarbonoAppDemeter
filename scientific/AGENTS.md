# Scientific AGENTS

- Fonte oficial ou literatura revisada deve ser registrada antes de usar qualquer parâmetro.
- Separe fato observado, declaração do usuário, dado derivado e inferência.
- Toda saída quantitativa precisa de unidade, intervalo, incerteza e versão do método.
- Não copie uma metodologia certificadora parcialmente e apresente como conformidade.
- Alterações exigem revisão humana documentada em `REVIEW_LOG.md`.
