# Escopo da implementação de referência

As imagens fornecidas são contratos visuais do fluxo principal, não capturas de um produto cientificamente certificado. O desenvolvimento deve chegar à mesma leitura visual e operacional, com estas exceções obrigatórias:

- textos deformados são substituídos por microcopy natural;
- valores de carbono sem metodologia ativa recebem `DEMO`, `Cenário demonstrativo` ou `Em cálculo`;
- nenhum estado `Elegível` equivale a certificação ou emissão;
- imagens rurais de demonstração devem ser claramente dados de amostra;
- controles devem ser acessíveis e responsivos.

O app é considerado visualmente concluído quando as seis capturas canônicas passam pelo checklist e os desvios restantes estão documentados no relatório de diff.
